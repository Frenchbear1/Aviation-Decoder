# Aviation Decoder

A lightweight, single-page aviation abbreviation decoder for METAR, TAF, Chart Supplement, and general aviation shorthand.

**Features**
- Decodes a wide range of METAR, TAF, Chart Supplement, and general abbreviations
- Simple one-field lookup with instant results
- Includes an info footer with usage guidance
- Live time display

**Usage note**
Paste or type *individual segments* of a METAR or TAF (e.g., one token at a time), not the entire report.
